# 🍕 Super Mario Pizza - Cadastro de Cliente

Bem-vindo ao projeto **Super Mario Pizza - Cadastro de Cliente**, um formulário simples e funcional para cadastro de novos clientes com validação de dados via JavaScript.

## 📋 Descrição

Este projeto tem como objetivo criar uma página de cadastro de cliente com um design limpo e agradável, seguindo boas práticas de estrutura HTML, estilização com CSS e validação com JavaScript puro.

## 📁 Estrutura de Pastas

```
projeto/
├── index.html
├── style.css
├── scripts/
│   ├── header.js
│   └── validaCadastro.js
├── styles/
│   ├── globalStyle.css
│   ├── header.css
│   └── footer.css
├── public/
│   ├── logo-mario.png
│   └── estrela-portuguesa.png
```

## 🚀 Funcionalidades

- Validação de campos obrigatórios 🟢
- Verificação de formato de e-mail 📧
- Verificação de senha mínima e confirmação 🔐
- Estilo responsivo com layout centralizado 📱

## 🧪 Validações

✔️ Todos os campos são obrigatórios\
✔️ O e-mail deve estar no formato válido (`exemplo@dominio.com`)\
✔️ A senha deve conter pelo menos 6 caracteres\
✔️ A confirmação da senha deve ser igual à senha digitada

> ❗ Caso alguma validação falhe, o formulário **não será enviado** e um alerta será exibido ao usuário.

## 🛠 Tecnologias Utilizadas

## 📝 Como usar

1. Clone o repositório:

   ```bash
   git clone https://github.com/seu-usuario/super-mario-pizza-cadastro.git
   ```

2. Abra o `index.html` em seu navegador:

   ```bash
   cd super-mario-pizza-cadastro
   start index.html
   ```

3. Preencha os campos do formulário e veja as validações em ação! 🚀
